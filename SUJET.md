# Starter — Projet de fin de cours (Développement d'application mobile, CESI)

Ce dépôt est un **starter** que vous pouvez utiliser à votre convenance pour réaliser votre **projet de fin de cours**. Vous partez de cette base Expo (React Native) et vous construisez votre application selon le cahier des charges definis en cours (écrans, API, navigation, etc.).

---

## Contenu du starter

- **Expo SDK 52** + **Expo Router** (routing par fichiers sous `app/`)
- **TypeScript** (mode strict), alias `@/*`
- **React Navigation** (Stack) ; possibilité d’ajouter des ongles (Bottom Tabs)
- **Dépendances** : Reanimated, Gesture Handler, Safe Area, Haptics, Splash, WebView, etc.
- **Plateformes** : iOS, Android, Web
- **Écran actuel** : un seul écran d’accueil (`app/index.tsx`) en placeholder — à remplacer ou étendre selon votre projet

---

## Démarrage

1. Cloner le dépôt (ou récupérer le ZIP).
2. `npm install`
3. `npx expo start`
4. Ouvrir dans Expo Go (QR code), simulateur iOS/Android, ou navigateur (web).

**Scripts :** `npm run android`, `npm run ios`, `npm run web`

---

## Utilisation pour votre projet

Vous êtes libres d’utiliser ce starter comme base, de le modifier, d’ajouter des écrans, une API, une navigation plus riche, etc. La structure `app/` et les dépendances sont là pour éviter de repartir de zéro ; le reste dépend de votre projet de fin de cours.
